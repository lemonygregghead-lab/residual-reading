# Residual Reading: Tender Buttons Under Pressure

This project is an interactive reading space for *Tender Buttons* in which attention acts on the text. Rather than producing stable interpretations, it generates shifting textual states through interaction—hovering, clicking, and duration—each of which introduces a different form of instability.

As the text changes, it leaves behind “residues”: traces shaped by contact, time, and repetition. These residues do not preserve an original version but record what reading does over time.

The project treats attention as active, embodied, and generative. Meaning does not resolve; it accumulates, disperses, and returns.

Try interacting with each phrase differently and notice how the text—and your reading—changes.
